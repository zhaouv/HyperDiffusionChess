# HyperDiffusionChess

自创规则的棋

Hyper Diffusion: 双方轮流放置棋子, 不能放置在4个以及以上的相邻的棋子旁边. 当放置的棋子构成多于等于4个棋子相邻时, 该玩家必须继续放置棋子. 须放置却没有空位置的玩家输.

目前状况: 实现了 本地对战和网络对战, {3,7}镶嵌的双曲棋盘

待实现: AI对手, 9路围棋棋盘, 蜂巢棋盘, 三角棋盘

- "Square"
- "Hexagonal"
- "Quadrilateral"
